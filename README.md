# Gadget Renesas Arduino Tool Chain for Linux

WIP（作業中）

## 概要
IDE for GRのLinux版が提供されないので代わりに作りました。
Arduino IDEに読み込ませて使います。~/Arduino/hardwareフォルダに展開してください。

## 制限事項
- GR-LYCHEEのみ対応しています。
- platform.txt の tools.savesketch.upload.pattern は修正してください。
